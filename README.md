# The Odin Project - Full Stack JavaScript

## #14 Project: Battleship

A Battleship game made with Webpack, Biome, Jest, HTML, CSS, JS.

*Our first TDD (Test-driven development) project!*

### Features

- PVE Mode
- PVP Mode

> Note: PVP Mode can be played locally with another person by taking turns on the same device.
